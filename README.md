# Flipkart Clone

This project is a simple frontend clone of **Flipkart** built using **HTML, CSS, JavaScript and Bootstrap**.  
It focuses on layout, visuals and responsiveness.  
No backend or login system is included.

## Features

- Flipkart-style homepage  
- Product cards with images, title, price and rating  
- Carousel banners using Bootstrap  
- Category sections  
- Responsive design that adjusts for mobile and desktop  
- Navigation bar with search box and basic icons  


## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **Bootstrap 5**

## How to Run

Download or clone the project:

```bash
git clone https://github.com/jvkedev/Flipkart-Clone.git
```

Open index.html in your browser.

## Notes

- This is a static frontend project.
- No backend, database, authentication or API calls are used.
- The goal is to replicate the UI for learning and practice.
